# АРМ Сотрудника дознания

Автоматизированное рабочее место сотрудника дознания для работы с карточками розыска (КРД).

## Структура проекта

- `main.py` - основной файл запуска приложения
- `authorization.py` - модуль авторизации пользователей
- `login_window.py` - окно авторизации
- `main_window.py` - главное окно приложения с отображением КРД
- `krd_details_window.py` - окно просмотра и редактирования данных КРД
- `add_krd_window.py` - окно добавления новой КРД
- `migration_request_tab.py` - вкладка формирования запроса в миграционную службу
- `admin_user_management_tab.py` - административная вкладка управления пользователями
- `init_db.py` - файл инициализации базы данных

## Требования

- Python 3.8+
- PyQt6
- psycopg2-binary (для PostgreSQL)
- bcrypt

Установка зависимостей:
```bash
pip install PyQt6 psycopg2-binary bcrypt
```

## Настройка базы данных

1. Установите PostgreSQL
2. Создайте базу данных согласно структуре в файле "Ход выполнения создания базы данных.docx"
3. Убедитесь, что пользователь `arm_user` существует с паролем `ArmUserSecurePass2026!`

## Запуск приложения

1. Инициализируйте базу данных:
```bash
python init_db.py
```

2. Запустите приложение:
```bash
python run_app.py
```

## Учетные данные по умолчанию

- Администратор: `admin` / `admin123`

Для добавления новых пользователей используйте административную вкладку в главном окне приложения.

## Функциональность

- Авторизация пользователей (регистрация осуществляется администратором в отдельной вкладке)
- Просмотр списка КРД
- Добавление новых КРД
- Редактирование данных КРД
- Формирование запросов в миграционную службу
- Поиск по фамилии, имени, отчеству и номеру КРД
- Управление пользователями (только для администраторов)
